# LP設計書 — AIキャリア診断サービス

## 概要

「5問でわかる、あなたのキャリア」のランディングページ設計書。
Stage1ではLPのみを実装し、認証・診断機能はStage2以降で段階的に追加する。

## セクション詳細設計

### 1. ヒーローセクション（HeroSection）

| 項目 | 内容 |
|------|------|
| コンポーネント | `app/components/HeroSection.tsx` |
| 背景 | linear-gradient（ブルー → インディゴ → パープル） |
| 高さ | 100vh（ビューポート全体） |
| レイアウト | 中央揃え、縦方向に要素を配置 |

#### テキスト内容

| 要素 | テキスト | スタイル |
|------|---------|---------|
| キャッチコピー | 「5問で見つかる、あなたの未来。」 | h2, 白, bold, 2.5rem〜3.5rem |
| サブコピー | 「就活・転職・キャリアチェンジ。AIがあなたの経験と適性を分析し、最適なキャリアロードマップを提案します。」 | body1, 白（0.85透過）, 1rem〜1.25rem |
| 補足テキスト | 「登録不要 · 完全無料 · 約1分で完了」 | caption, 白（0.7透過）, 0.875rem |

#### CTAボタン

| 項目 | 仕様 |
|------|------|
| ラベル | 「無料で診断スタート」 |
| スタイル | 白背景、テキスト色はプライマリ、角丸16px、padding大きめ |
| ホバー | scaleアニメーション + boxShadow強調 |
| Stage1動作 | Snackbarで「Coming Soon! 現在準備中です」を表示 |

#### レスポンシブ

| ブレークポイント | 調整 |
|----------------|------|
| sm以下 | タイトル 2rem、サブコピー 0.95rem、パディング縮小 |
| md以上 | タイトル 3rem、maxWidth: 800px |
| lg以上 | タイトル 3.5rem |

| 項目 | 内容 |

#### セクション見出し

| 要素 | テキスト |
|------|---------|
| 見出し | 「SurvibeAIの特徴」 |

#### カード（3枚）

| # | アイコン | タイトル | 説明 |
|---|---------|---------|------|
| 1 | AutoAwesome (MUI Icon) | AIがあなた専用のロードマップを作成 | 汎用的なアドバイスではなく、あなたの回答に基づいたパーソナライズされたキャリア提案を行います。 |
| 2 | Timer (MUI Icon) | たった5問、約1分で完了 | 長い診断は不要。厳選された5つの質問に答えるだけで、精度の高い分析結果が得られます。 |
| 3 | Balance (MUI Icon) | 中立的で客観的な分析 | 転職サイトやエージェントとは異なり、AIが偏りのない視点であなたのキャリアを分析します。 |

#### カードスタイル

| 項目 | 仕様 |
|------|------|
| 形状 | MUI Card、角丸 16px |
| 影 | elevation: 0、ホバー時 elevation: 4 |
| アイコン | 48x48、プライマリカラー背景（透過10%）、丸背景内 |
| ホバー | translateY(-4px) + shadow |

#### セクション見出し

| 要素 | テキスト |
|------|---------|
| 見出し | 「かんたん3ステップ」 |

#### ステップ（3つ）

| Step | アイコン | タイトル | 説明 |
|------|---------|---------|------|
| 1 | QuizOutlined | 5つの質問に回答 | 簡単な選択式の質問に答えていきます。所要時間は約1分。 |
| 2 | Psychology | AIが回答を分析 | あなたの回答をAIが多角的に分析し、キャリアの方向性を導き出します。 |
| 3 | Map | ロードマップを受け取る | あなた専用のキャリアロードマップが完成。具体的な次のステップが見えてきます。 |

#### ステップ間のコネクタ

- md以上：ステップ間に矢印アイコン（→）を表示
- sm以下：非表示
