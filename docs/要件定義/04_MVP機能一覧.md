## LPコンテンツ設計

### ヒーローセクション

| 要素 | 内容案 |
|------|-------|
| キャッチコピー | 「5問で見つかる、あなたの未来。」 |
| サブコピー | 「就活・転職・キャリアチェンジ。AIがあなたの経験と適性を分析し、最適なキャリアロードマップを提案します。」 |
| CTAボタン | 「無料で診断スタート」 |
| 補足テキスト | 「登録不要 · 完全無料 · 約1分で完了」 |

### サービス特徴（3つ）

| # | タイトル | 説明 |
|---|---------|------|
| 1 | AIがあなた専用のロードマップを作成 | 汎用的なアドバイスではなく、あなたの回答に基づいたパーソナライズされたキャリア提案を行います。 |
| 2 | たった5問、約1分で完了 | 長い診断は不要。厳選された5つの質問に答えるだけで、精度の高い分析結果が得られます。 |
| 3 | 中立的で客観的な分析 | 転職サイトやエージェントとは異なり、AIが偏りのない視点であなたのキャリアを分析します。 |

### 診断の流れ（3ステップ）

| ステップ | タイトル | 説明 |
|---------|---------|------|
| Step 1 | 5つの質問に回答 | 簡単な選択式の質問に答えていきます。所要時間は約1分。 |
| Step 2 | AIが回答を分析 | あなたの回答をAIが多角的に分析し、キャリアの方向性を導き出します。 |
| Step 3 | ロードマップを受け取る | あなた専用のキャリアロードマップが完成。具体的な次のステップが見えてきます。 |

### 下部CTA

| 要素 | 内容案 |
|------|-------|
| 見出し | 「あなたのキャリア、次の一歩を踏み出そう。」 |
| CTAボタン | 「無料で診断スタート」 |
| 補足テキスト | 「5問に答えるだけ · 登録不要 · 完全無料」 |

## コンポーネント設計（Next.js + MUI）

### ページ構成

```
app/
├── layout.tsx          # 共通レイアウト（MUI ThemeProvider）
├── page.tsx            # LPトップページ
├── components/
│   ├── HeroSection.tsx       # ヒーローセクション
│   ├── FeaturesSection.tsx   # サービス特徴セクション
│   ├── StepsSection.tsx      # 診断の流れセクション
│   ├── CtaSection.tsx        # 下部CTAセクション
│   ├── Footer.tsx            # フッター
│   └── common/
│       ├── CtaButton.tsx     # 共通CTAボタン
│       └── SectionTitle.tsx  # セクション見出し
├── theme/
│   └── theme.ts              # MUIカスタムテーマ
└── lib/
    └── constants.ts          # テキスト・定数定義
```

### MUIテーマ方針

| 項目 | 方針 |
|------|------|
| プライマリカラー | 信頼感のあるブルー系（#1976D2近辺） |
| セカンダリカラー | アクセントとなるコーラル/オレンジ系 |
| タイポグラフィ | Noto Sans JP（日本語）、Inter（英数字） |
| スペーシング | 8pxグリッドシステム |
| ブレークポイント | xs: 0, sm: 600, md: 900, lg: 1200, xl: 1536（MUIデフォルト） |

## 非機能要件

### パフォーマンス

| 指標 | 目標値 |
|------|--------|
| LCP（Largest Contentful Paint） | ≤ 2.5秒 |
| FID（First Input Delay） | ≤ 100ms |
| CLS（Cumulative Layout Shift） | ≤ 0.1 |
| Lighthouse Performance Score | ≥ 90 |

### SEO

| 項目 | 対応内容 |
|------|---------|
| title | 「5問でわかる、あなたのキャリア｜AI キャリア診断」 |
| meta description | サービス概要を120文字以内で記載 |
| OGP | og:title, og:description, og:image を設定 |
| 構造化データ | WebApplication スキーマ（JSON-LD） |
| サイトマップ | next-sitemap で自動生成 |

### アクセシビリティ

| 項目 | 対応内容 |
|------|---------|
| WCAG | 2.1 Level AA 準拠を目標 |
| カラーコントラスト | 4.5:1 以上を確保 |
| キーボード操作 | 全CTAがTab/Enterで操作可能 |
| スクリーンリーダー | 適切なaria-label、見出し階層 |
| 画像 | alt属性を必ず設定 |
