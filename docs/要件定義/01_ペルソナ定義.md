# ペルソナ定義

## 概要

AIキャリア診断サービス「5問でわかる、あなたのキャリア」のターゲットユーザーを3タイプのペルソナとして定義する。

---

## ペルソナ1：就活迷子の大学生 — 田中 美咲（22歳）

### 基本属性

| 項目 | 内容 |
|------|------|
| 年齢 | 22歳 |
| 職業 | 大学4年生（文系・経営学部） |
| 居住地 | 東京都内（一人暮らし） |
| デバイス | iPhone（メイン）、ノートPC |
| 情報収集 | X（旧Twitter）、Instagram、YouTube、就活系メディア |

### 状況・背景

- 就活解禁後、エントリーシートを20社以上出したが、軸が定まっていない
- 「やりたいこと」が見つからず、周囲が内定を取り始めて焦っている
- 自己分析は何度もやったが、結果を活かしきれていない
- OB/OG訪問もしたが、話を聞くほど選択肢が増えて混乱している

### ペインポイント

- 自己分析ツールは多いが、結果が抽象的でアクションに繋がらない
- 「向いている職種」だけでなく、具体的なキャリアパスが知りたい
- 長い診断は途中で離脱してしまう

### ニーズ

- 短時間で自分の適性がわかる具体的な診断
- 「次に何をすべきか」が明確にわかるロードマップ
- 同世代の成功事例やリアルな情報

### サービスに期待すること

- 5問で完結する手軽さ
- AIならではの客観的・データドリブンな診断結果
- 診断結果から具体的な業界・職種・行動ステップが見える

### LP上での行動パターン

- SNSの広告やシェアから流入
- スマホで閲覧、スクロールして全体を俯瞰
- 「無料」「5問だけ」のハードルの低さに惹かれてCTAを押す

## ペルソナ2：転職検討中の若手社会人 — 佐藤 健太（27歳）

### 基本属性

| 項目 | 内容 |
|------|------|
| 年齢 | 27歳 |
| 職業 | IT企業 法人営業（社会人5年目） |
| 居住地 | 神奈川県（実家暮らし） |
| デバイス | iPhone、会社支給PC |
| 情報収集 | X、NewsPicks、転職サイト、ビジネス系YouTube |

### 状況・背景

- 新卒で入社した企業で5年目、仕事には慣れたがマンネリを感じている
- 同期が転職して年収アップした話を聞き、自分も転職を意識し始めた
- ただし、今の会社を辞める決定的な理由もなく、踏み切れない
- 転職サイトに登録したが、求人を見ているだけで行動に移せていない

### ペインポイント

- 転職すべきか、現職に留まるべきか判断軸がない
- 営業以外のキャリアも考えたいが、何ができるのかわからない
- 転職エージェントに相談すると転職を勧められる一方向のアドバイスになりがち

### ニーズ

- 転職の是非を含めた中立的なキャリア診断
- 自分のスキルや経験から可能なキャリアチェンジの選択肢
- 3年後・5年後のキャリアビジョンを描く手助け

### サービスに期待すること

- 転職エージェントではない中立的な立場からの分析
- 現職での成長可能性も含めた多角的な診断
- AIによる客観的かつパーソナライズされた結果

### LP上での行動パターン

- Google検索「キャリア診断 無料」や転職系記事から流入
- PCまたはスマホの通勤時間帯に閲覧
- サービスの信頼性（AI活用、実績）を確認してからCTAを押す

## ペルソナ3：キャリアの方向転換を模索する30代 — 山田 彩（31歳）

### 基本属性

| 項目 | 内容 |
|------|------|
| 年齢 | 31歳 |
| 職業 | メーカー勤務 マーケティング担当（社会人9年目） |
| 居住地 | 大阪府（パートナーと同居） |
| デバイス | iPhone、自宅のMacBook |
| 情報収集 | Voicy、Podcast、note、LinkedIn |

### 状況・背景

- 30代に入り、管理職を目指すか専門性を深めるか悩んでいる
- 結婚・出産などライフイベントも視野に入り、働き方を見直したい
- 副業やフリーランスにも興味があるが、リスクが気になる
- これまでのキャリアを棚卸しして、次の10年を設計したい

### ペインポイント

- 年齢的に「次の一手」の重要度が高く、失敗したくない
- キャリアコンサルタントへの相談は費用と時間のハードルが高い
- 汎用的なアドバイスではなく、自分の状況に合った具体策が欲しい

### ニーズ

- ライフステージを踏まえた長期的なキャリア設計
- 管理職・専門職・独立など複数のキャリアパスの比較
- 気軽に始められるが、深い洞察が得られる診断

### サービスに期待すること

- ライフイベントも考慮したキャリアロードマップ
- 管理職/専門職/独立の各パスのメリット・デメリット提示
- 5問で終わる手軽さと、AIならではの精度の高い分析の両立

### LP上での行動パターン

- noteやLinkedInの記事、口コミから流入
- 夜間の自宅でじっくり閲覧
- サービスのロジックや信頼性を重視し、「診断の流れ」を読んでからCTAを押す

## ペルソナ比較サマリー

| 観点 | 田中 美咲（22歳） | 佐藤 健太（27歳） | 山田 彩（31歳） |
|------|-------------------|-------------------|-----------------|
| ステージ | 就活中 | 転職検討中 | キャリア再設計 |
| 主な悩み | やりたいことが見つからない | 転職すべきか判断できない | 次の10年をどう設計するか |
| 求める結果 | 適職・業界の提示 | キャリアチェンジの選択肢 | ライフステージ込みのロードマップ |
| 行動特性 | 手軽さ・無料に反応 | 信頼性・中立性を重視 | 論理性・深さを重視 |
| 主な流入経路 | SNS広告 | 検索・転職メディア | 口コミ・ビジネスメディア |
| LP上の関心 | CTAの手軽さ | サービスの信頼性 | 診断ロジックの説明 |
